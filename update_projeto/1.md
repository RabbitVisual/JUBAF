## 1. O Fluxo da "Ata Digital" (Secretaria 2.0)

Em vez de apenas um campo de arquivo, implementaremos um fluxo de trabalho (Workflow):

1.  **Redação:** O Secretário redige a ata diretamente no sistema (usando um editor *Rich Text*).
2.  **Revisão:** A ata fica em status "Rascunho" para correções.
3.  **Assinatura Digital (Interna):** * O Presidente e o Secretário recebem uma notificação.
    * Eles assinam usando sua **senha de login** ou um **PIN de segurança** único.
    * O sistema registra: *ID do Usuário, IP, Data/Hora exata e Hash do Documento.*
4.  **Fechamento:** Uma vez assinada, o sistema gera um PDF final com um **QR Code de Verificação** no rodapé.
5.  **Publicação:** A ata é movida automaticamente para o "Acervo" e fica disponível para as 70 igrejas (conforme o nível de permissão).

## 2. Implementação Técnica para o Módulo `Secretaria`

Para profissionalismo total: **Nenhuma despesa extraordinária no módulo Financeiro deve ser aprovada sem o link para a Ata que a autorizou.** Isso cria uma trilha de auditoria que a diretoria anterior não tinha.
